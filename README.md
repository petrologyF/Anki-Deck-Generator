# Anki Deck Generator

JSON形式の単語リストから、高品質なAnkiデッキ（.apkgファイル）を自動生成するためのデスクトップツールです。VS Codeをリスペクトしたプロフェッショナルなデザインで、効率的な学習をサポートします。

---

## 🚀 概要

このアプリは、学生や研究者が「カード作成」ではなく「学習」に集中できるように設計されています。使い慣れたJSON形式でデータを管理し、ボタン一つで美しいデザインのカードを作成できます。

### ✨ 主な機能
- **VS Code風UI**: ダークモード標準対応の、スタイリッシュな制作環境。
- **ワークフロー誘導**: 次にすべき操作（保存・生成）をアイコンの「青い光」が教えてくれます。
- **洗練されたデザイン**: Ankiの標準カードよりも見やすく、品詞や例文が際立つCSSを内蔵。
- **Discord送信機能**: 生成されたデッキを自動でDiscordに送信し、スマホなど他デバイスでの受け取りをスムーズにします。

### 🛠️ セットアップ（初心者向け）
1. **Pythonの準備**: PCに [Python 3.8以上](https://www.python.org/downloads/) がインストールされていることを確認してください。
2. **ダウンロード**: このリポジトリ全体をダウンロードして解凍します。
3. **ライブラリのインストール**: フォルダ内でターミナル（PowerShell等）を開き、以下を実行します：
   ```bash
   pip install -r requirements.txt
   ```
4. **実行**: 
   ```bash
   python run.py
   ```

### 📖 基本的な使い方
1. **編集**: 中央のエディタに単語データを入力します。
2. **保存**: 左端の **保存（ディスク）アイコン** をクリックします（変更があると青く光ります）。
3. **生成**: **生成（再生）アイコン** をクリックします（保存が完了すると青く光ります）。
4. **受取**: `output` フォルダにデッキが保存されます。Discord設定済みなら自動送信されます。
